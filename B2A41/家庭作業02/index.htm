SWOT分析，三階段重點一把抓

SWOT是什麼？要如何使用呢？

SWOT分析是一種由美國舊金山大學的管理學教授韋里克提出的分析法，
許多公司會運用在行銷策略、產品的規劃上。
俗話說：「知己知彼，百戰百勝。」，
SWOT分析這四個字分別由：
優勢（Strengths）、劣勢（Weaknesses）、機會（Opportunities）和威脅（Threats）
等元素組成，可以幫助企業同時觀照自身內部與外部的條件，幫助釐清思考方向。
SWOT分析長久以來被廣泛而大量的運用，但許多人常常誤把它當做了魔術帽，
認為自己放進去的任何東西都可以轉化成盛開的鮮花，而這種期待是非常危險而錯誤的。
SWOT分析本身仍只是一個工具，重點是你如何靈活並正確的使用它。

SWOT分析前：釐清目標，確認方向

首先，你需要先定義好自己的目標，確認好要討論的範圍，SWOT分析並非幫助你無中生有，而是幫助你釐清一些可能的選項。
接著，你也可以順道思考你的Product-Market Fit（PMF）-「產品市場媒合度」，將你的目標客群與產品定位納入討論當中，而不是只是憑空虛無的進行分析。
最後，在真正開始進行SWOT分析之前，要讓所有參與者都充分了解你們的目標，當大家站在同一條軌道上時，才能讓後續的資料搜集與討論連成一線。

SWOT分析中：避免含混不清，重質不重量

針對優勢（Strengths）、劣勢（Weaknesses）、機會（Opportunities）和威脅（Threats）
等四個面向，要用精確的方式來回答，不是將表格填的越滿越好，而是深入研究、大量搜集資料，了解並歸納出最符合這些面向的要素。
往內探尋你的優勢（Strengths）、劣勢（Weaknesses），同時往外查看你的機會（Opportunities）和威脅（Threats）。
在這四個面向中你可以利用以下的問題進行發想：

  • 優勢（Strengths）面向
    1. 公司有什麼專業或特色是競爭對手沒有的？
    2. 公司熟悉什麼樣的東西/領域？
    3. 公司有什麼樣的資源？
    4. 公司能做到什麼別人做不到或比別人好的？
  • 劣勢（Weaknesses）面向
    1. 公司特別不熟悉什麼？
    2. 公司在什麼方面的資源比較少？
    3. 公司比較做不到什麼？
  • 機會（Opportunities）面向
    1. 有什麼樣的服務或產品是公司可以發展的？
    2. 現在的產業環境對公司而言有什麼可能？
  • 威脅（Threats）面向
    1. 競爭對手可能會如何影響公司？
    2. 產業環境的變動會不會造成公司的不利？

SWOT分析後：延伸運用與限制

不要以為做完SWOT分析就沒事了，要好好利用你辛苦分析出來的結果才能發揮它最大的價值！
但要如何思考與延伸呢？我們提供另外一個模型—「USED」給你參考。
USED分別代表的是Use、Stop、Exploit、Defend

  • 而這四個面向與SWOT當中的四個元素結合就成為：
    1. 如何善用每個優勢？ How can we Use each Strength?
    2. 如何停止每個劣勢？ How can we Stop each Weakness?
    3. 如何成就每個機會？ How can we Exploit each Opportunity?
    4. 如何抵禦每個威脅？ How can we Defend against each Threat?
從以上的問題出發，將你從SWOT分析出的結果發展成為更具體的行動，實際規劃並動手執行。

  • 此外除了分別檢視每個面向之外，你也可以將它們兩兩相配一起思考：
    1. SO - 如何使用你的優勢（Strength）與機會（Opportunity）？
    2. ST - 如何使用優勢（Strength）同時避免威脅（Threats）？
    3. WO - 如何降低弱勢（Weaknesses）並運用機會（Opportunities）？
    4. WT - 如何降低弱勢（Weaknesses）與避免威脅（Threats）？

SWOT本身的限制


  • 同個產業類別分析出的結果同質性高
    1. 在相同的領域當中，因為大家面臨的環境狀況差異不大，可能遇到的威脅以及目前所掌握的技術也相差不遠，所以導致很多企業進行SWOT分析之後，會產生非常類似的結果，沒辦法凸顯解決問題的方式。
  • 分析規劃與實際行動之間的時間差
    1. 使用SWOT分析時，考慮的時間段落主要以現在能搜集到訊息為主，未來的發展較難預測，也因此當規劃完畢開始行動時，四個面向的元素可能又因為時間變化而有所更動，影響到你原先設想的應對方式。

結語

除了運用在企業層面的思考之外，SWOT也可以用於個人層面的分析，瞭解你自身的優缺點、機會與威脅等。
而公司若是希望能夠更為全面的進行思考，也可以搭配其他的分析法：例如PESTEL、波特五力等，幫助你更為深入的探尋自身企業內外部的資訊。
光有工具是還不夠的，要規劃長遠策略需要一步步將基礎紮穩紮深，而集客式行銷能夠幫助你統整這些工具並全面性的應用，讓你更為有架構的進行發展。
